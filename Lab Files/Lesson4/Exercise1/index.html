My first web page

What's it all about?

This is a webpage to demonstrate some of the most useful elements (also known as tags) that are available in HTML. You can use it as a starting point for adding in HTML of your own. You can also use it to practise using CSS to add styling and JavaScript to add functionality.

Some of the things you should know about HTML

  • There is a always a head and there is always a body
    1. The head is used to add "meta" infomation
    2. The head can contain links to separate CSS and JavaScript files
    3. The body is where you put all the HTML elements that are to be displayed on the page
  • Elements have two parts - an open and a close
  • You can use F12 (Developer Tools) to inspect the HTML in this page
